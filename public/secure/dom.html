The Fix

🔴 Before (Vulnerable)
// ❌ innerHTML — parses HTML el.innerHTML = `Hello, ${name}!`;
🟢 After (Secure)
// ✅ append() — Text nodes only el.replaceChildren(); el.append( 'Hello, ', name, '!' );

Safe APIs vs Dangerous Sinks — Reference

🔴 Dangerous — parse HTML
el.innerHTML = input; el.outerHTML = input; document.write(input); el.insertAdjacentHTML ('...', input); eval(input); setTimeout(input, 0); location.href = input;
🟢 Safe — Text nodes
el.textContent = input; el.append(input); el.setAttribute( 'data-x', input); // When HTML needed: el.innerHTML = DOMPurify.sanitize(input);
